Magenta
 This group specifies what OS the plut is for. Do not edit it
Blue
 This specifies the bit depth of the resource. the default is fine
Red
 unknown. always 002A
Yellow
 How many color sets are in the resource.  the number must be converted to Hex.  The 100 above is, in regular decimal numbers, a 256
Cyan
 This is a color set.  The dark part tells what ppat should be used for this brush.  take the ID of the ppat, convert it to HEX and enter it here.  to use no ppat, enter a 0000 here
the light part specifies the color that will be used if no ppat is entered.  For this part of the resource, select the color you want in your favorite graphics app and open the apple color picker. select the Web section, and uncheck the "Snap to web color" checkbox.  then click on the "old" color.  write down the hex number.  Then enter it "doubled" into the resource.  A color of
AB CD EF would be entered in this section as a ABAB CDCD EFEF.
Green
 Same as the cyan row:
dark is the ppat Id in Hex
and light is the color
Grey
 Each row is another color set like blue and green.
