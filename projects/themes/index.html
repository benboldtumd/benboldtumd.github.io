These are all the Mac OS Classic themes that I made in years past.
Compatible with Mac OS 8.5 - 9.2.2. Kaleidoscope NOT required.

To see how I made these themes, visit this page. It describes all of the details I found by experimenting with each type of resource in the Macintosh resource forks of themes.


Click the icons below to download the themes.

AquaMaX AquaMaX makes your Mac look much like it has Mac OS X. This is the best Mac OS X theme availble.
Doohickey 2.0 Doohickey is a goofy theme that looks similar to Apple's beta-released "Gizmo" theme and it shows off some things that themes can do that can't be done in Kaleidoscope. I finally finished this theme after it sat unfinished for a very long time. If you play around enough, you will discover some very interesting functionality.
Classic DSX Classic DSX was relatively easy for me to create from the existing Mac OS X theme. Because I made AquaMaX so accurate, I was pretty much able to just copy the images over. Very few layout changes needed to be made. Much of my time was used converting images with transparency masks (nearly ALL of them) into opaque images, and trying to maintain the original transparent look.
Moby Moby is a theme like no other. It has BeOS-style windows, and everything is a bizarre organic shade of green. The menus have a stretching gradient background, something not generally thought of as being possible to do. To this day, I use this theme in the Classic environment of my Mac OS X system.
Classic Sosumi Sosumi is similar to DSX in that it is based on a Mac OS X theme. Likewise, I made very few layout changes. If you want to convert a Mac OS X theme into a classic theme, this would be the theme to choose because it has extremely organized resources.
