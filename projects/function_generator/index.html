Function Generator


This project was intended to create a function generator, both in simulation and physically, with the following characteristics:
  • Square wave output
  • Triangle wave output
  • Sine wave output
  • Sawtooth wave output
  • Adjustable frequency
  • Adjustable amplitude
I was able to implement this however I wanted, with any components available in the lab. The output was not required to be perfect, but it was expected to be a good approximation. This page shows only Pspice simulation images, but the physical circuit operated true to the simulation as well. No images were taken of the physical implementation at the time because it was only for demonstration to the lab assistant, after which it was disassembled.

I started with an astable multivibrator circuit. This produces a square wave output. The frequency is also easily adjusted with a potentiometer. Also, the voltage across the capacitor very closely resembles a triangle wave. I then passed both of these two waveforms through unity gain buffers so that any load would not affect the multvibrator.

Triangle Wave

Next, I integrated the triangle wave to approximate a sine wave. This was a straightforward approach. It doesn't produce a perfect sine wave, but that's all we need here.



Sine wave
Finally, I had to produce a sawtooth wave, which posed a real problem for me. I was not sure how to derive a sawtooth wave from the other waves I have. But there is a way. I added the triangle wave to the square wave, then took the absolute value. This produces a sawtooth wave with twice the frequency.


Square wave added to triangle wave


Absolute value of square + triangle

All waves simulated at the same time demonstrates that the waves are synchronous, but the amplitudes vary. Amplitudes were not required to be equal, so this output still meets all requirements.
